Replace Property of directives in Angular

Angular directives has replace property which actually used to replace directive tag from DOM. This property is actually helpful when we have some elements has some css that is interdependent. Consider following example:-

angular
   .module('mangroosHub')
   .directive('notificationList', notificationList);

   notificationList.inject = [''];
   function notificationList() {
     // Usage:
     //
     // Creates:
     //
     var directive = {
         bindToController: true,
         templateUrl: 'app/components/navbar/user-dropdown/notification-list/notification-list.html',
         controller: NotificationListController,
         controllerAs: 'notificationList',
         link: link,
         replace: false,
         restrict: 'E',
         scope: {
         }
         };
       return directive;
} 

                        -----notification-list.html-----
<ul class="nav navbar-nav navbar-right acme-navbar-text">
    <notification-list></notification-list>
    <li class="dropdown">
        <a ng-href="#" class="dropdown-toggle">
            <i class="fa fa-envelope" aria-hidden="true></i>
            <span class="label">10</span>
        </a>
        <ul class="dropdown-menu">
            <li class="header">You have 4 messages.</li>
            <li><a href="#">Support Team</a></li>
            <li><a href="#">Admin Team</a></li>
        </ul>
    </li>
</ul>
                        

                        ----notification.css-----
.nav .navbar-nav > li.dropdown > a {
	//Some css rules here
}

In the above example, If we set replace property false(default value is also false) inside var directive, then in our html we have <notification-list></notification-list> element in DOM which results into breaking css for our HTML. In order to fix this, we have to set replace property true which results into replacing element <notification-list></notification-list> from DOM, resulting into only having html of directive.

After setting replace true in var directive we have following HTML,

                        -----notification-list.html-----
<ul class="nav navbar-nav navbar-right acme-navbar-text">
    <li class="dropdown">
        <a ng-href="#" class="dropdown-toggle">
            <i class="fa fa-envelope" aria-hidden="true></i>
            <span class="label">10</span>
        </a>
        <ul class="dropdown-menu">
            <li class="header">You have 4 messages.</li>
            <li><a href="#">Support Team</a></li>
            <li><a href="#">Admin Team</a></li>
        </ul>
    </li>
</ul>
